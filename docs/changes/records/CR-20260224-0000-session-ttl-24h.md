# CR-20260224-0000-session-ttl-24h

## 1. 变更意图（Compact Summary）

- 背景：当前会话保持时间为6小时，部分场景需要更长的离线保留时间
- 目标：将 SESSION_IDLE_TTL_MS 从 21600000（6小时）修改为 86400000（24小时）
- 本次边界：仅修改配置值，不涉及代码逻辑变更

## 2. 实施内容（What changed）

1. 修改 `.env.example` 中 `SESSION_IDLE_TTL_MS` 默认值从 21600000 改为 86400000
2. 更新 REQ 文档中相关描述

## 3. 影响范围（Files/Modules/Runtime）

- 模块：会话管理配置
- 运行时行为：会话空闲保留时间从6小时延长至24小时
